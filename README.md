## What is Dryfish License?

The Dryfish License is a permissive open-source license that grants users freedom to use, modify, and distribute software, while protecting the original author's brand through mandatory renaming of modified versions.

## Why Dryfish?

- ✅ Freedom to use in proprietary/commercial software
- ✅ Simple attribution requirement
- ✅ Brand protection via mandatory renaming
- ✅ Automatic termination for violations
- ✅ Compatible with existing open-source ecosystem

## Quick Start

```bash
# Clone the repository
git clone https://github.com/yourname/dryfish-license

# Use in your project
cp licenses/Dryfish-1.0.txt /path/to/your/project/LICENSE
```
